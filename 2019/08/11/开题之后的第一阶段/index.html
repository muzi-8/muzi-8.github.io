开题之后的第一个阶段

发表于 2019-08-11 | 分类于 科研学术

开题到现在也有两个月之久(2019.6.11-2019.8.11)，原计划是要参加创新工场举办的Deecamp夏令营，研究一个对抗样本的课题，给指导老师说明情况之后，建议自己还是把重心放在自己的课题上面，所以就没有如期参加，现在回头想想，也没有遗憾。记录一下这两个月的工作进展以及不足之处。

课题进展

深度学习作为现在主流的方法之一，但是其最大的诟病就是“black-box”，只给出用户最终结果，但是缺少了原因，着严重阻碍了以Deep-learning为基础的人工智能的发展。

肯定之处：

  1. 与指导老师讨论三次，推动课题的第一部分进行。（时间间隔：4周、6周、8周）

  2. 文献书写：参考书籍《Academic Phrasebank》，增强文献书写的逻辑性

  3. 课题补充：参考书籍《Interpretable Machine Learning 》， A guide for making black box models explainable

  4. 代码方面：对“解释器”Explainer开源代码的整理与实验。(注：所有的代码都是在pytorch框架下)

1） MASK 优化方法：jacobgil/pytorch-explain-black-box

2） 根据网络激活值方法：kazuto1011/grad-cam-pytorch / Gradcam++

3） 依据梯度方法：yulongwang12/visual-attribution / kazuto1011/grad-cam-pytorch / utkuozbulak/pytorch-cnn-visualizations

不足之处：实验整体进度慢。

动力支撑

在课题研究的过程中，难免会出现疲惫与焦虑，这两个月给我动力的三大来源是：

1）实验室邀请的黄博士进行的科研内容分享与建议（参考：赛博智能公众号：新闻纪实|百度AI研究员黄家骥博士应邀报告ACL2019最新研究）

2）作为志愿者参与清华大学举办的线下交流分享活动“AI-Time”（详见本博客文章：Explainable Machine Learning）

3）(正在进行当中) 阅读吴军博士最新书籍《全球科技通史》，对于科学与技术有了一个全新的认识，对于自己的科研更有了敬畏之情。

生活

与她相处三个月了，有快乐时刻，有拌嘴时刻，这大概就是爱情吧。（承诺给她写的code一直搁置）

不足之处：由于时间有限，与她相处的时间多了，搁置了自己的健身计划

其他

1）申请Deecamp之前，刷了一半的《数据结构》并没有坚持下去。

2）线上花钱报名的流利英语说，也没有坚持给搁置了。

3）Lihongyi 课程刷了一半也放弃了LeeML-Notes
