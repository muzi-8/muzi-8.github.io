文献UnderstandingtheErrorStructureasaKeytoRegularize ConvolutionalNeuralNetworks解读

发表于 2018-09-09 | 分类于 文献解读

摘要

在分类任务中，由于样本之间具有高度相似性，导致类别之间经常相互混淆。这种相似性使得类别之间存在粗细(coarse-to-fine)粒度(granularity、fine-grained)的层次结构。作者开发了一个可视分析系统用来揭示这种结构，并且分析了此结构现象是怎么出现在模型的训练过程当中。

作者发现：

  • 在训练的过程当中，少量的epoch，网络就可以将物体大致分到宽泛地类别组当中，后续的epoch，就是使得物体分为更细粒度的类别中。

  • 对应到网络结构当中，就是网络模型的前几层学到的特征能够实现粗分类（coarse classification），网络后面的几层就是学到更专业特征然后实现细粒度的分类。

作者进一步扩展了Alexnet网络，在网络的不同层引入了额外损失函数去较快实现分类任务。最终作者通过可视分析手段发现的一些insight,能够起到加速收敛的作用，并且使得在ALexnet分类器提升了网络的性能，Imagenet数据集的Top-1错误提升了20%，Top-5的错误提升了30%

引言

相关工作

理解卷积网络的行为模式已经有大量地工作，大部分工作聚焦于可视化网络学习到的特征，利用的手段与途径大致分为以下几类：

  • perturbation

  • deconvolution

  • code inversion

  • activation maximization

这些可视化出来的图像作用：

  • 明确训练集中存在的偏差(biases)，例如:哑铃图像总包含着手臂。
  • 帮助选择卷积核参数（filter parameters）
  • 评估滤波器的冗余性

不足：可视化的特征图像不能够提供模型的行为概览。

本文焦点

很少有工作关注分类错误这个问题，典型的方式就是通过“线图（Line Graph）”去监控某个分类器训练过程中的错误率或者去比较多个分类器的性能, 通过“直方图（Histogram）”的方式表示预测得分之间错误的相关性[1,2,3]。

“混淆矩阵（confusion matrices）”这种形式能够提供更多的误分类信息。所以作者的整个可视分析系统工作也是基于混淆矩阵展开的。
