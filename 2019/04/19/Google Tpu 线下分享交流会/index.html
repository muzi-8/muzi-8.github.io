TPU分享交流

忙里偷闲，参加了线下TPU分享交流，实际上自己对于TPU之前一无所知，因为对于我们而言，无需太关注硬件设计，只要用好硬件资源即可。现对TPU做一简单的科普介绍

  • TPU（tensor processing unit）专门为Google的深度学习框架tensorflow框架而设计。

    Google的AI芯片

  • cpu 包括运算器（ALU，Arithmetic and Logic Unit）和控制器（CU,Control Unit）两大部件。CPU从程序计数器取到指令，通过指令总线将指令送至译码器，将转译后的指令交给时序发生器与操作控制器，然后运算器对数据进行计算，通过数据总线将数据存至数据缓存寄存器。CPU遵循的是冯诺依曼架构，核心就是：存储程序，顺序执行。但是不足之处在于效率不是很高，能否将多个处理器放在同一块芯片上呢？

  • GPU（Graphics Processing Unit）图形处理器，当前最顶级的CPU只有4核或者6核，模拟出8个或者12个处理线程来进行运算，但是普通级别的GPU就包含了成百上千个处理单元。虽然GPU是为了图像处理而生的，但是它还被用来进行科学计算、数值分析等任务，因此被认为是一种较通用的芯片。

  • FRGA（Field Programmable Gate Array）运用硬件描述语言描述逻辑电路，即一种可以编程的芯片。

  • TPU(Tensor processing unit)一种ASIC(专用集成电路)，一款专用机器学习算法的专用芯片。TPU的核心是脉动阵列，如下图所示：
