Explainable Machine Learning

发表于 2019-07-03 | 分类于 深度学习

Understand How and Know Why

Motivation

In recent years, convolutional neural networks (CNN) are state of the art when it comes to computer vision tasks, sucn as image recognition and object detection. However, due to the high nonlinearity of these networks, it is notoriously difficult to understand predictions made by these networks. The most problematic issue of interpretability is to define explanable rules.

Interpretability Goals

There are at least three reasons for why interpret CNN. They are that understanding is just as important as performance; sensitive applications(medical or military) require a thorough understanding of its inner workings; CNNs can be improved if their predictions are properly understood.

Besides, there are the following four main goals:

  • General Understanding
  • Architecture Assessment
  • Model Quality Assessment
  • User Feedback Integration

Related Work

To help end users better understand a deep CNN, researchers in the field of computer vision have made efforts to interpret their predictions and illustate the learned feature of each neuron. The available methods for interpreting CNNs can be broadly grouped into two categories, namely, attribution methods and visualization methods.

Attribution Methods

Attribution methods attributes a classification result back to the input pixels, meanwhile, output is usually a heatmap over input pixels that indicates how much each pixel contributed to the classification outcome. Heatmap highlight the relative importance of areas of the input image for predictions. Furthermore, the mainstream approaches about attribution methods are divided into three types, such as perturbation based approaches, gradient based approaches and relevance score approaches.

Visualization Methods

Visualization methods aim to look at the learned parameters of the neural network and visualize the patterns that filters have learned. Existing methods can be classified into two categories.:code inversion and activation maximization. Code inversion methods synthesize an image from the activation vector of a specific layer, which is produced by a real image. Activation maximization methods aim to find an image that maximally activates a given neuron, which is modeled as an optimization problem over the image space.
